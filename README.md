### How to run

- Download Go+ and build it. See https://github.com/goplus/gop#how-to-build.
- Download this game and run.
  * git clone https://github.com/goplus/FlappyCalf.git
  * cd FlappyCalf
  * ./run.sh

- Download GoMobile and build it. See golang.org/x/mobile.
- Build Mobile
  * ./buildmobile.sh
